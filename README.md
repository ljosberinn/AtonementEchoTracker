# Atonement / Echo Tracker

... that also tracks Rejuvation on Restoration Druid.

## What It Does

Shows an icon with the next expiration time of whichever hot you provide.

Additionally renders the amount of players its applied to.

## Features

- supports:
  - Preservation Evoker > Echo
  - Restoration Druid > Rejuvation
  - Discipline Priest > Atonement
- Edit Mode integration
  - resizing
  - positioning
  - opacity
  - icon zoom
  - font
  - font flags
  - duration font size
  - stack font size
  - default state
  - duration color
  - stack color
  - border style
  - show duration fractions
  - hide mask
  - stack count anchor
  - stack count offset
  - load conditions
